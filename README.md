# Java Calculator

A command-line calculator application built with Java.

## Features
- Basic operations: Addition, Subtraction, Multiplication, Division
- Advanced operations: Power, Square Root, Modulus, Percentage
- Input validation and error handling
- User-friendly menu interface

## How to Run
```bash
javac Calculator.java
java Calculator
```

## Technologies Used
- Java
- Scanner for user input
